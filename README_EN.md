## Introduction

This is a Chinese resume LaTeX template specifically designed for students at The Hong Kong University of Science and Technology (Guangzhou). It adopts the university's official color scheme and provides professional, clean resume formatting.

## Preview

The template features the official HKUST(GZ) blue theme (RGB: 0, 71, 122), including decorative header and footer bars, university logo watermark, and other visual elements that present a professional academic resume style.

## Key Features

- ‚úÖ **Official Color Scheme**: Uses HKUST(GZ) official blue theme
- ‚úÖ **No Image Dependencies**: Header and footer drawn with TikZ code, no manual background image creation needed
- ‚úÖ **Font Compatibility**: Supports both Overleaf and local TeX distributions with automatic font adaptation
- ‚úÖ **Rich Icons**: Integrated FontAwesome5 icon library

## Quick Start

### Compilation Requirements

- **Compiler**: XeLaTeX (Required)
- **Recommended Platforms**:
  - Online: [Overleaf](https://www.overleaf.com/)
  - Local: VS Code + LaTeX Workshop extension

### Compilation Steps

1. Clone or download this repository
2. Compile `main.tex` using XeLaTeX
3. Generate PDF resume file

## Customization

### Changing Colors

Find the color definition in `settings.tex`:

```latex
\definecolor{HKUST_GZ}{RGB}{0,71,122}
```

Modify the RGB values to change the theme color. Examples:
- Dark blue: `{0, 51, 102}`
- Dark red: `{139, 0, 0}`

### Modifying Section Titles

Find the corresponding `\section{...}` command in `main.tex` and modify the text content.

Example:
```latex
\section{\makebox[0.4em][c]{\faLaptopCode}\quad Projects \& Internships}
```

### Changing Icons

The template uses the FontAwesome5 icon library. Browse icon codes at: [FontAwesome Icon Library](https://fontawesome.com/v5/search?m=free)

Common icon examples:
- `\faGithub` - GitHub
- `\faEnvelope` - Email
- `\faPhone` - Phone
- `\faPython` - Python
- `\faLaptopCode` - Programming

## FAQ

### Q1: Icons not displaying?

**A**: Ensure FontAwesome5 package is properly loaded. Check if `settings.tex` contains:
```latex
\usepackage{fontawesome5}
```

### Q2: Compilation error "missing font"?

**A**: This template has optimized font compatibility and defaults to Fandol font family. If issues persist, please verify:
- Using XeLaTeX compiler
- TeX distribution updated to latest version

### Q3: How to adjust header and footer height?

**A**: Find the TikZ drawing code in `settings.tex` and modify the corresponding coordinate parameters.

### Q4: Can this be used for English resumes?

**A**: Yes, but font settings need adjustment. An English-specific template is recommended for better results.
